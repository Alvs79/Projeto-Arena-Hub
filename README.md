# Projeto-Arena-Hub

Integrantes: Enzo Abreu Aramuni Alves, Paulo Henrique da Silva Ribeiro, Caio Mendes Carvalho, Rafael Coimbra dos Santos, Rafael Furtado Magalhães, Caio Fernandes Lima;

Front-end: HTML, CSS e Java Script - Back-end: Python e PHP;

  O ARENA HUB é uma plataforma Vertical SaaS e Marketplace voltada para a gestão e conexão de arenas de esportes de areia. O sistema centraliza reservas, pagamentos, gestão financeira e organização de torneios, oferecendo uma experiência mais eficiente tanto para proprietários quanto para atletas.
  Além da operação das arenas, a plataforma inclui recursos de validação de nível técnico, ranking unificado entre torneios, divisão automática de pagamentos e ferramentas de engajamento social, como compartilhamento de highlights e feed de vídeos da comunidade.
  O objetivo do projeto é profissionalizar o ecossistema dos esportes de areia, reduzindo processos manuais, aumentando a ocupação das arenas e criando uma experiência digital integrada para jogadores e organizadores;
